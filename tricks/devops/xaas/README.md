## FAQ

### tenant vs user - 租户和用户的区别
* Tenant - 租户
  * 从服务商以租赁的方式获取服务
* Vendor
  * 个体或企业以批量的形式进行服务售卖
* Customer - 顾客
  * 实际购买主体
* User - 用户
  * 使用某服务的任意实体
