# PEG.js

## Tips
* [pegjs.org](https://pegjs.org) - JavaScript PEG 语法解析生成器
  * [在线](https://pegjs.org/online) / [文档](https://pegjs.org/documentation)
* 语法
  * `''` - 单引号字符串忽略大小写
  * `""` - 双引号字符串大小写敏感
  * `!` `&` - 用于条件预测 - 类似于前向环视 - Lookahead
* [metadevpro/ts-pegjs](https://github.com/metadevpro/ts-pegjs)
  * 生成 TS 源码
